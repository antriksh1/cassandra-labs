Lab : Nodetool

Nodetool command is used to administer a C* cluster
nodetool can be found in  CASSANDRA_INSTALL_LOCATION/bin


== STEP 1) invoke nodetool
    $   nodetool
or
    $   /path/to/cassandra/install/bin/nodetool

This will print out all commands


== STEP 2) get status of cluster
hint : use 'status'  or 'describecluster' arguments to nodetool


== STEP 3)  find / explore a 'cool' nodetool command
