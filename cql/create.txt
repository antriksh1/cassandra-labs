== STEP 1)  Invoke cqlsh
    $  cqlsh
you should see the prompt
    cqlsh>

==  STEP 2) Create a keyspace 'myflix'
Enter the following in cqlsh
    cqlsh>
        CREATE KEYSPACE myflix
        WITH REPLICATION = {
            'class' : 'SimpleStrategy',
            'replication_factor' : 1
            };

    cqlsh>
        describe keyspace myflix;
